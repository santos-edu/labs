# Template de PRD

Este diretório contém o template base para criar um novo laboratório.

## Uso

1. Copie `PRD.md` para o novo diretório de laboratório.
2. Atualize o título para `# PRD: YYYY-MM-Tema`.
3. Preencha as seções de objetivo, problem statement, tarefas, critérios de sucesso, recursos e riscos.

## Estrutura do template

- `Objetivo` — qual problema o laboratório resolve.
- `Problem Statement` — contexto do problema e impacto.
- `Tarefas` — atividades necessárias.
- `Critérios de Sucesso` — definição de sucesso do experimento.
- `Recursos` — links e referências úteis.
- `Riscos e Limitações` — o que deve ser considerado.
